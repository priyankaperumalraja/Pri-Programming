Basics of C++

How do I start writing code?

When starting out as a programmer, all the complicated wording, syntax, and theory in computer science can be incredibly daunting and discouraging.

Here are some of the basic code formatting you'll need for almost every project:

The Main Function

C++ is an Object-Oriented Programming language (To refresh what this means, go back to the Types of Programming Languages lesson).

Essentially, this means that code or "instructions" are written using objects. We'll cover objects in a later lesson but for now, this just means that code is usually separated based on its purpose in the program.

This is the case for the main function or the block of code that holds all the main code in your program. Its purpose is to run the main code, so it's categorized into its own function.

EXAMPLE

How a main function looks like in C++:

FILE NAME: example.cpp

#include <iostream>

using namespace std;

int main() {

cout << "Hello World!";

return 0;

}

NOTE: In most cases, you'll ALWAYS need a main function in the format listed above, so get used to typing it out!

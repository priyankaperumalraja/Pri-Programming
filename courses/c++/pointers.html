Pointers

What is a Pointer?

Pointers are arguments that "point" to the address of specific variables.

Doesn't this sound a lot like reference variables? Yes!

While pointers and reference variables both hold addresses of variables, there are some key differences to each:

Pointers

References

Address it holds can change.

Can be assigned the NULL variable (a variable that represents nothing).

To access value at the address, must “dereference” the pointer (Essentially add a star before the name of the pointer).

Address it holds is ONLY what it references (for example, a reference variable called &item can ONLY hold the address of the variable item).

CANNOT be assigned the NULL variable, must contain an address.

No need to explicitly dereference, just state the name of the variable without the ampersand (&).

Pointers AND References

Can be passed through functions as arguments.

Can hold the address of arrays.

Pointers & Arrays

Pointers can also point to arrays!

For a pointer pointing to an array, the default value it holds is the address of the first array element:
